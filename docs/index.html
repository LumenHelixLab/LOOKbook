lookBOOK is an open-source book-to-animation compiler. Upload a comic page or archive, and it extracts panels, OCRs text, tracks characters, builds scene graphs, and exports to every major video platform.

End-to-End Pipeline

From raw comic pages to platform-ready animation exports in a single command.

🗄

Panel Detection

OpenCV-based contour detection segments pages into individual panels, ordered by Western reading direction (top→bottom, left→right).

📝

OCR Extraction

Tesseract-powered text extraction with comic-specific preprocessing. Classifies text as dialogue, narration, SFX, or caption automatically.

👤

Character Tracking

Perceptual hashing tracks character appearances across panels and pages, building consistency sheets for animation handoff.

🎬

Scene & Shot Graphs

Scene graphs group panels by dialogue continuity and character presence. Shot graphs generate timed camera moves, transitions, and motion directives.

🎤

M7 Choreography Pipeline

build-choreography links OCR dialogue to panels and speakers. Shot graphs emit one shot per choreography line; living-panels HTML and CineForge ingest bridge the review handoff.

🌐

Multi-Platform Export

Export to Runway ML, Veo 2, Kling, Pika, Luma, ComfyUI, FFmpeg, and Remotion — each with platform-optimized prompts and workflows.

📦

Batch Archive Processing

Process entire CBZ/CBR comic archives in one command. Extracts every page, runs the full pipeline, and produces all exports.

Data Flow

How a comic page becomes an animation export.

Source Image ➔ Panel Detection ➔ OCR Text ➔ Characters ➔ Scene Graph ➔ Choreography ➔ Shot Graph ➔ Exports
# Full pipeline on a single page
lookbook detect-panels source.png my_project/
lookbook extract-text source.png my_project/
lookbook extract-characters source.png my_project/
lookbook build-scene-graph my_project/
lookbook build-choreography my_project/
lookbook build-shot-graph my_project/
lookbook export-living-panels my_project/
lookbook export-cineforge my_project/
# Or batch process an entire archive
lookbook process-archive my_comic.cbz my_project/

CLI Reference

21 commands across scaffolding, source intelligence, choreography review, and platform exports.

Scaffolding

  • lookbook init → Create a new lookBOOK project
  • lookbook analyze-source → Analyze a source file
  • lookbook demo → Create a demo project
  • lookbook install-demo-lab → Install the demo lab

Source Intelligence

  • lookbook detect-panels <source> <project> → Detect comic panels
  • lookbook extract-text <source> <project> → OCR text extraction
  • lookbook extract-characters <source> <project> → Extract character clusters
  • lookbook build-scene-graph <project> → Group panels into scenes
  • lookbook build-shot-graph <project> → Generate timed animation breakdown (one shot per choreography line)
  • lookbook build-choreography <project> → Speech-driven choreography from OCR + panels

Choreography & Review

  • lookbook export-living-panels <project> → Self-contained browser living-panels HTML review
  • lookbook export-cineforge <project> → CineForge storyboard ingest JSON (optional --push)

Platform Exports

  • lookbook export-runway <project> → Runway ML workflow
  • lookbook export-veo <project> → Veo 2 / Gemini prompts
  • lookbook export-kling <project> → Kling, Pika & Luma prompts
  • lookbook export-comfyui <project> → ComfyUI workflow JSON
  • lookbook export-ffmpeg <project> → FFmpeg assembly scripts
  • lookbook export-remotion <project> → Remotion TypeScript project

Batch & Archive

  • lookbook list-pages <archive> → List pages in CBZ/CBR
  • lookbook process-archive <archive> <project> → Full pipeline on all pages

M7 Choreography Pipeline

Speech-driven review between scene grouping and shot timing — bridges lookBOOK analysis to CineForge living review.

🎤

build-choreography

Links dialogue and narration to panels, bubble bboxes, inferred speakers, and word lists for browser SpeechSynthesis boundary sync. Output: analysis/choreography.json.

🎥

Per-Line Shot Graph

When choreography exists, build-shot-graph emits one timed shot per choreography line with camera moves derived from active dialogue — not just per-scene fallbacks.

🖼

export-living-panels

Self-contained HTML review with panel overlays, voice cast, and line-by-line playback. Default: exports/living_panels/review.html.

🌐

export-cineforge

Validated shot graph + choreography ingest for CineForge storyboard shots. File export by default; --push --project-id for live API handoff.

# M7 choreography → living review → CineForge bridge
lookbook build-choreography my_project/
lookbook build-shot-graph my_project/
lookbook export-living-panels my_project/
lookbook export-cineforge my_project/
lookbook export-cineforge my_project/ --push --project-id <cineforge-uuid>

See CINEFORGE_BRIDGE.md for API details and ingest payload schema.

Get Started

Install lookBOOK and start processing comic pages in minutes.

Prerequisites

  • Python 3.10+
  • Tesseract OCR (optional, for text extraction) — choco install tesseract or your system's package manager
  • OpenCV + numpy (optional, for panel detection) — installed automatically with the vision extras

Install

# Clone and install
git clone https://github.com/your-org/lookBOOK.git
cd lookBOOK/lookBOOK_true_animation_github_repo

# Create virtual environment
python -m venv .venv
source .venv/bin/activate # or .venv\Scripts\activate on Windows

# Install with all features
pip install -e ".[image,vision,archive,dev]"

Quick Start

# Process a single comic page
lookbook init my_project/
lookbook analyze-source page.png my_project/
lookbook detect-panels page.png my_project/
lookbook extract-text page.png my_project/
lookbook build-scene-graph my_project/
lookbook build-shot-graph my_project/
lookbook export-remotion my_project/

# Or batch an entire archive
lookbook process-archive comic.cbz my_project/
